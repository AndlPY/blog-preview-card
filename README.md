# Практична робота №3 — Blog Preview Card

## Опис проєкту

Цей проєкт є практичною роботою №3 з теми створення картки блогу за наданим дизайном.

Мета роботи — реалізувати компонент картки блогу за допомогою HTML та CSS, використовуючи знання з Flexbox і Grid.

## Реалізовано

- Створено HTML-структуру сторінки.
- Реалізовано окремий компонент картки блогу.
- Додано зображення, категорію, дату публікації, заголовок, опис і блок автора.
- Використано CSS для стилізації картки.
- Налаштовано відступи, закруглення кутів, тінь і вертикальне розташування контенту.
- Проєкт завантажено на GitHub.
- Проєкт підготовлено до публікації через GitHub Pages.

## Технології

- HTML5
- CSS3
- Flexbox
- Git
- GitHub
- GitHub Pages
